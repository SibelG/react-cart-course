
### Technologies Used :
  * React-JS
  * React Hooks

  
### How to Run Locally :

1. Clone the Repository
  
     `git clone https://https://github.com/SibelG/react-cart-course`

2. cd into Events-Console
  
      `cd react-cart-course`
      
3. Install all dependencies
      
      `npm install`
      
4. Start Local Server
      
      `npm start`
